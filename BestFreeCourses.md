# Best Free Courses

A collection of free resources to learn computer science and programming.

## Computer Science Courses

* [Open Source Society University (OSSU) Computer Science](https://github.com/ossu/computer-science): A comprehensive curriculum for learning computer science.

## Data Structures and Algorithms

* [MIT Lectures: Data Structures and Algorithms](https://youtube.com/playlist?list=PLUl4u3cNGP61Oq3tWYp6V_F-5jb5L2iHb): Video lectures from MIT on data structures and algorithms.
* [MIT Lectures: Data Structures and Algorithms (PDFs)](https://drive.google.com/drive/folders/1UpzcAC0-bCn6zgfLLFacFscyInBqy6-y?usp=sharing): PDF notes from the MIT lectures.

## Must-Read Books

* [The Pragmatic Programmer](https://drive.google.com/file/d/1TJkwSSxCeQpz-__kGailNGhDnM4vqAbF/view?usp=sharing): A must-read book for programmers.

## CS50 Lectures

* [CS50 Lectures 2020](https://youtube.com/playlist?list=PLhQjrBD2T382_R182iC2gNZI9HzWFMC_8): Video lectures from Harvard's CS50 course.

## Behavioral Interviews

* [Prepare for Behavioral Interviews](https://www.youtube.com/channel/UCrtPXCqjm9c7nqgJL-vBSZg): A YouTube channel with tips and resources for acing behavioral interviews.
